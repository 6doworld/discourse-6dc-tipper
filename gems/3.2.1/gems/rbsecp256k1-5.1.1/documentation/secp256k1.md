Secp256k1
=========

Secp256k1 is the top-level module for this library.

Class Methods
-------------

#### have_recovery?

Returns `true` if the recovery module was built with libsecp256k1, `false`
otherwise.

#### have_ecdh?

Returns `true` if the EC Diffie-Hellman module was built with libsecp256k1,
`false` otherwise.
